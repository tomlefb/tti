## Caveat — terminal cap vs broker archive

If every instrument shows ~99999 bars, that is almost certainly the MT5 terminal's *Max bars in chart* setting (default 100000), not the true broker archive depth. To verify the broker's actual depth, raise the cap in the terminal:

1. In MetaTrader 5: **Tools -> Options -> Charts -> Max bars in chart** -> set to a large value (e.g. 5000000 or 'Unlimited').
2. Restart the terminal so the new cap takes effect.
3. For each watched symbol, scroll the M5 chart all the way to the left (or press End / Home) so the terminal pulls the full history from the broker into local cache.
4. Re-run this script.

Until that is done, the depth measured here is a **lower bound** on what FundedNext can actually provide.
